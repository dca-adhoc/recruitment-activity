# Desafío de Ingeniería - Gestión de Ciudadanos

Este repositorio contiene la resolución al desafío de procesamiento de datos de ciudadanos, aplicando buenas prácticas de Programación Orientada a Objetos en Python.

## Características
* Modelado de la entidad `Persona` de forma inmutable y con validación de datos.
* Procesamiento mediante la clase `RegistroPersonas` para cálculo de métricas (promedio, extremos de edad y segmentación).
* Acceso eficiente (O(1)) para consultar edades mediante DNI.

## Cómo ejecutar el script

Para correr este proyecto en tu entorno local, sigue estos pasos:

1. Asegúrate de tener instalado Python 3.
2. Abre una terminal en la carpeta raíz del proyecto.
3. Ejecuta el siguiente comando:
   ```bash
   python3 activity.py
